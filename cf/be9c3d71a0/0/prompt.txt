In `ChangesMinimap.tsx`, the minimap line buttons use `transition-all duration-150` alongside a conditional `w-2.5` / `w-3.5` toggle. `transition-all` will animate the `width` property change, which is not GPU-accelerated.
**CLAUDE.md rule:** *"Animate only `transform` and `opacity` — never `width`, `height`, `top`, `left`, `margin`, `padding`."*
**Fix:** Replace `transition-all` with `transition-opacity` and use `scale-x` transform for the width effect, or just drop the width animation entirely.
