1. Cursor Move + Ripple (BrowserClick)

Cursor travels from current position to element center, then blue ripple ring expands. Used before click actions.

2. Cursor Dwell + Fade (after Click)

After click: cursor dwells 600ms then fades out over 250ms. Compare with instant hide.

3. Cursor Pin (BrowserType)

Cursor moves to input, pins to it (follows via rAF). Stays pinned until you click "Unpin".

4. Ripple Only

Blue expanding ring at click position. 22px, scale 0.8 -> 1.35, 180ms ease-out.

5. Screenshot Flash (BrowserScreenshot)

White overlay flash. Full viewport or region-scoped. opacity 0.35 -> 0 over 300ms.

6. Element Highlight (BrowserPressKey)

Blue outline + fill pulse on focused element. Stays 300ms, fades over 400ms.

7. Page Scan (BrowserSnapshot)

80px luminance band sweeps down viewport. ease-out-expo (800ms). Light passing over glass.
